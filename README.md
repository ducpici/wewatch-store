# 🛍️ Website Thương Mại Điện Tử — React + TailwindCSS + ExpressJS + TypeScript

E‑commerce full‑stack dùng **React (Vite)** + **TailwindCSS** cho frontend và **ExpressJS** cho backend. Sử dụng **MySQL** lưu trữ cơ sở dữ liệu.

> Mục tiêu: Codebase chuẩn, rõ ràng, dễ mở rộng, triển khai nhanh cho cửa hàng vừa & nhỏ.

---

## ✨ Tính năng

-   **Auth & phân quyền**: đăng ký/đăng nhập, JWT.
-   **Sản phẩm & danh mục**: CRUD danh mục, thuộc tính.
-   **Tìm kiếm & lọc**: theo từ khoá, danh mục, khoảng giá; sort theo giá.
-   **Giỏ hàng**: thêm, xóa, cập nhật sản phẩm.
-   **Voucher**: phần trăm/cố định, thời gian hiệu lực, giới hạn lượt.
-   **Thanh toán**: chuyển khoản ngân hàng + COD.
-   **Đơn hàng**: tạo → thanh toán → xử lý → vận chuyển → hoàn tất (timeline rõ ràng).
-   **Admin**: sản phẩm, đơn hàng, voucher, người dùng, dashboard doanh thu.

---

## 🧱 Công nghệ

-   **Frontend**: React 18 + Vite, TypeScript, TailwindCSS, Axios.
-   **Backend**: ExpressJS (JavaScript), Multer (upload).
-   **CSDL**: MySQL 8+.
